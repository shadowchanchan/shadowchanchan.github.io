众筹平台浅悟

发表于 2018-01-27 | 分类于 产品体验


体验网站：kickstarter，京东众筹，苏宁众筹，天涯众筹，一点时间

众筹是什么？

翻译自国外crowdfunding一词，即大众筹资或群众筹资，香港译作「群众集资」，台湾译作「群众募资」。由发起人、跟投人、平台构成。具有低门槛、多样性、依靠大众力量、注重创意的特征，是指一种向群众募资，以支持发起的个人或组织的行为。一般而言是透过网络上的平台连结起赞助者与提案者。群众募资被用来支持各种活动，包含灾害重建、民间集资、竞选活动、创业募资、艺术创作、自由软件、设计发明、科学研究以及公共专案等。
现代众筹指通过互联网方式发布筹款项目并募集资金。相对于传统的融资方式，众筹更为开放，能否获得资金也不再是由项目的商业价值作为唯一标准。只要是网友喜欢的项目，都可以通过众筹方式获得项目启动的第一笔资金，为更多小本经营或创作的人提供了无限的可能。

规则

  1. 筹资项目必须在发起人预设的时间内达到或超过目标金额才算成功。
  2. 在设定天数内，达到或者超过目标金额，项目即成功，发起人可获得资金；筹资项目完成后，支持者将得到发起人预先承诺的回报，回报方式可以是实物，也可以是服务，如果项目筹资失败，那么已获资金全部退还支持者。
  3. 众筹不是捐款，支持者的所有支持一定要设有相应的回报。

众筹平台的分类

  1. 产品众筹：又叫回报众筹。顾名思义，就是发起者筹款成功后交付的东西是个产品，可以是个实物，也可以是虚拟商品，还可以是个事件或者活动。一般以小额为主。 （京东众筹、淘宝众筹）
  2. 债权众筹：就是大家常说的P2P，不在本回答讨论范围。
  3. 股权众筹：支持者投入资金获得标的公司的股份或者收益权。一般起投金额较大，人数一般限定在200人以内，属于高风险的投资型众筹。 （开始众筹）
  4. 公益众筹：纯粹捐赠性质的不图回报的众筹。（轻松筹）

平台流程

发起人-发起项目-平台审核-平台发布-公开项目
支持者-进入平台-浏览项目-支持项目-等待众筹结果-获得相应价值
目前发起众筹流程大同小异。
共同点：人工审核机制+平台等级门槛
不同点：专注领域不同，京东，苏宁是小电器为主，天涯农产品，kickstars。

角色&需求

  • 发起人需求
    拥有更多元化展示方式展示自己的项目，及时与支持者沟通和反馈，能推广到自己的项目
  • 支持者需求
    多个项目选择，全面了解项目信息，提供安全，信任感，且支持项目进度可追踪
  • 平台职责
    审核项目真实有效性，管理好当前进行项目，做好项目结束后，退款，售后等对应服务。

发现的问题

  1. 除非有强大的平台背书，或者平台本身已经有客户资源，能够先吸引一些优质项目，吸引支持者，不然就导致平台项目少，质量差，无法让支持者放心。
  2. 纵观所有平台，都没有很强有力的监管机制，如果选择支持项目，很大部分还是得信任这个平台，不至于钱被坑。
  3. 国内的项目和国外的项目运营方式差别还是挺大的。国外项目的发起者不定期更新项目动态并且和支持者互动频密，给人感觉这个项目不断在前进，可掌控。反观国内项目发起人很少公布项目目前进展，互动的信息在平台上也没有明显的入口，国内更像换个噱头贩卖商品的团购频台，而没有真的是众筹项目的那种互动性。（kickstarter）
  4. 国内众筹都是垂直行业发起，没有一个综合性平台，可能和国内众筹氛围差异有关，国外已经有游戏，电影，音乐等众筹，国内还是以实物、服务众筹为主。
