如何在数据库查找到多列同时重复的数据？

发表于 2019-01-04 | 分类于 编程 | 阅读次数:

问题

最近工作需要使用数据库查询重复的数据记录，核心数据难点是多列同时重复。

思路

看了不少网上资料，分组查询是主流。
最终确定思路是以两个字段同时分组把重复项查找出来，先用ORDER BY进行分组，再用COUNT(字段)>1进行筛选。
这种方法能够找到那些数据是发生了重复，但是需要知道重复了多少次，所以会在SELECT里加上数量的统计。
